# GV300 Coursework Portfolio

This repository contains my coursework for GV300, focusing on quantitative analysis in political science.

## Overview

In this project, I analyze survey and experimental data to study topics such as:
- A/B testing and Average Treatment Effects (ATE)
- Measurement and reliability (Cronbach’s alpha)
- Missing data and survey design issues
- Text analysis of open-ended responses

## Contents

- **Assignment 3** – A/B testing and treatment effects  
- **Assignment 5** – Measurement, reliability, and missing data  
- **Assignment 6** – Text analysis of survey responses  

## Methods Used

- Difference-in-means estimation  
- Linear probability models (`lm`)  
- Logistic regression (`glm`)  
- Reliability analysis (Cronbach’s alpha)  
- Basic text analysis (tokenization, word frequency)
